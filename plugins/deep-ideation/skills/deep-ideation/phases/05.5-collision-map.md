# Phase 5.5: COLLISION MAP

## Context Intake

Before starting, load these files in full:

| File | What to extract |
|------|----------------|
| `$WORKSPACE/05-john-a.md`, `05-john-b.md`, etc. (all John files) | Full transform outputs |
| `$WORKSPACE/01-discover.md` | TRIZ Contradiction Card, root causes |
| `$WORKSPACE/session-state.md` | Cohort IDs, open tensions |

## Output Header

Begin your output file (`$WORKSPACE/05.5-collision-map.md`) with:

```
## Upstream References
- `$WORKSPACE/05-john-*.md` — all John transform outputs (full)
- `$WORKSPACE/01-discover.md` — TRIZ card, root causes
- `$WORKSPACE/session-state.md` — cohort IDs
```

## DB Write Receipt

This phase does not add ideas to the DB. Return at the end:
```json
{"delta": 0, "ids": [], "updated": []}
```
Then append to `$WORKSPACE/session-state.md`:
```
Phase 5.5 (COLLISION MAP): completed, delta=0, ids=[], upstream=[05-john-*.md, 01-discover.md]
```

Launch the Collision Map agent. See `agents/collision-map.md`.

**Runs in:** STANDARD and DEEP modes only. Skip in LITE.

## When It Runs
After Phase 5 (TRANSFORM) — all three Johns have finished.
Before Phase 5.7 (RATCHET) — must complete before the Ratchet can run.

## What It Receives
- All Johns' complete outputs (`$WORKSPACE/05-john-a.md`, `05-john-b.md`, etc.)
- The Digger's TRIZ Contradiction Card and root causes (`$WORKSPACE/01-discover.md`)

## What It Produces
- Zone table: every sub-problem classified as HOT / WARM / COLD
- Hot zone details: Side A, Side B, core question, TRIZ connection
- Routing plan: HOT → Ratchet, WARM → Tension Analyzer, COLD → Brainwriter

Save to `$WORKSPACE/05.5-collision-map.md`.

## Mode Behavior
| Mode | Behavior |
|------|----------|
| LITE | Skip entirely |
| STANDARD | Run. Cap at 2 hot zones maximum. |
| DEEP | Run. All hot zones processed (max 3 by agent rules). |

## Tell the User

"Collision Map complete:
- [N] HOT zones → routing to Dialectical Ratchet
- [N] WARM zones → routing to Tension Analyzer later
- [N] COLD zones → passing through to Brainwriter"

## Anti-Patterns
- **Don't skip the Collision Map** — cold zones tell you where the skill is being boring; hot zones tell you where the best ideas hide
